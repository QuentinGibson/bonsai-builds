This guide defines the complete visual transformation from the warm amber/parchment aesthetic of BonsaiBuilds to the cool teal/slate aesthetic of Ethereal Navigator. Content, layout structure, and component hierarchy remain unchanged — only color, font, and surface treatment are replaced.

02 / Typography
Type System
Replace Georgia / condensed serif with Rajdhani for display/UI text, and Share Tech Mono for labels and stat readouts. Both have the angular, technical character needed for the blue-slate aesthetic.
Role Before (BonsaiBuilds) After (Ethereal Navigator)
Display / Headings Georgia, serif · bold 'Rajdhani', sans-serif · 700
UI Labels monospace · tracking 0.06em 'Share Tech Mono' · tracking 0.15em
Body / Secondary system-ui · regular 'Inter' · 300-400
Stat Numbers Georgia · color: #C8952A Rajdhani 700 · color: --accent-primary or --color-ascendancy

03 / CSS Variables
Design Token Reference
Drop this :root block into your global stylesheet. These replace all inline warm-amber color values from BonsaiBuilds.
:root { /* ── Backgrounds ─────────────────────────── */ --bg-canvas: #080C12; /* was #0D0B09 */ --bg-surface: #0E1520; /* was #15110D */ --bg-panel: #111927; /* sidebar bg — was #1A1410 */ --bg-card: #151F2E; /* stat cards — was #1E1A14 */ --bg-elevated: #1D2A3A; /* hover/elevated surface */ /* ── Accent ───────────────────────────────── */ --accent-primary: #00D4B4; /* was #C8952A (amber) */ --accent-hover: #00EDCA; --accent-glow: rgba(0, 212, 180, 0.15); --accent-border: rgba(0, 212, 180, 0.35); /* ── Text ─────────────────────────────────── */ --text-primary: #E8EEF5; /* was #E8D9B5 (warm white) */ --text-secondary: #7A8FA8; /* was #8A7355 */ --text-label: #4A6580; /* uppercase label color */ --text-muted: #3D5168; /* deemphasized text */ --text-accent: #00D4B4; /* interactive / highlight */ /* ── Semantic values ──────────────────────── */ --color-positive: #00D4B4; /* stat weight positive */ --color-neutral: #3D5168; /* stat weight ±0 */ --color-negative: #E85D5D; /* stat weight negative */ --color-ascendancy: #A87BE8; /* ascendancy stat number */ /* ── Borders ──────────────────────────────── */ --border-subtle: rgba(0, 212, 180, 0.08); --border-default: rgba(0, 212, 180, 0.15); --border-strong: rgba(0, 212, 180, 0.35); --border-panel: rgba(255, 255, 255, 0.05); /* ── Tree nodes ───────────────────────────── */ --node-default: #1E3D56; /* was #3D2A14 */ --node-active: #00D4B4; /* was #C8952A */ --node-connector: #00B89C; /* connector line between nodes */ /* ── Typography ───────────────────────────── */ --font-display: 'Rajdhani', 'Trebuchet MS', sans-serif; --font-mono: 'Share Tech Mono', 'Courier New', monospace; --font-body: 'Inter', system-ui, sans-serif; /* ── Spacing / Shape ──────────────────────── */ --radius-sm: 4px; --radius-md: 6px; --radius-lg: 10px; --sidebar-width: 180px; --panel-width: 300px; }
Google Fonts import
/* Add to <head> or @import in global CSS */ @import url('https://fonts.googleapis.com/css2?family=Rajdhani:wght@400;500;600;700&family=Share+Tech+Mono&family=Inter:wght@300;400;500&display=swap');
